# Wayfinder

Version: Pre-1.0
Current Phase: UI Polish Sprint

---

## Project Overview

Wayfinder is a Chrome Extension for Canvas LMS that provides a modern progress dashboard for students and instructors.

Primary goals:

- Make required coursework immediately visible.
- Allow instructors to define required assignments.
- Provide accurate progress tracking using the Canvas API.
- Feel like a native Canvas feature.
